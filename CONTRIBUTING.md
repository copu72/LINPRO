# Contribuir a LINPRO

1. Revisa docs/specifications/022_REGLAS_IA.md para entender las reglas de desarrollo.
2. Lee el ADR correspondiente a la decisión técnica que quieras implementar.
3. Código limpio, tipado, documentado.
4. Tests obligatorios en 	ests/unit/, 	ests/integration/, 	ests/regression/.
5. Pull request a la rama develop.
