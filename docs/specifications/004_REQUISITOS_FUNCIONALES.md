# 004 — REQUISITOS FUNCIONALES

**Versión:** 0.0.1  
**Fecha:** 2026-07-09

## Funcionales (alta prioridad)

1. **RF-01 Definición de alineaciones:** El usuario debe poder definir alineaciones a partir de puntos, ángulos, radios y parámetros de clotoides.
2. **RF-02 Sistema de PK:** El sistema debe calcular automáticamente los PK de cualquier punto del alignment.
3. **RF-03 Buffer:** El sistema debe generar un buffer alrededor del alignment con una distancia configurable.
4. **RF-04 Afecciones catastrales:** El sistema debe identificar las parcelas catastrales dentro del buffer.
5. **RF-05 Afecciones municipales:** El sistema debe identificar los municipios atravesados por el alignment.
6. **RF-06 Afecciones viarias:** El sistema debe identificar carreteras y caminos que intersectan el buffer.
7. **RF-07 Afecciones hidrográficas:** El sistema debe identificar ríos y arroyos que intersectan el buffer.
8. **RF-08 Afecciones de infraestructuras:** El sistema debe identificar líneas eléctricas, tuberías, etc. existentes.
9. **RF-09 Exportación Excel:** El sistema debe generar un informe Excel completo con todas las afecciones.
10. **RF-10 Exportación DWG:** El sistema debe generar un plano DWG con el eje, buffer y afecciones.
