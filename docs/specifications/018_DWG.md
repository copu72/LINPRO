# 018 — DWG / DXF

**Versión:** 0.0.1
**Fecha:** 2026-07-09

## Biblioteca

- **ezdxf** — Generación de archivos DXF.
- **DXF** como formato primario (estándar abierto). DWG como exportación secundaria (cuando sea posible).

## Capas del plano

| Capa | Color | Contenido |
|------|-------|-----------|
| `EJE` | 1 (rojo) | Línea del alignment |
| `PK` | 2 (amarillo) | Marcas de PK con texto |
| `BUFFER` | 3 (verde) | Polígono del buffer |
| `MUNICIPIOS` | 4 (cian) | Límites municipales |
| `CATASTRO` | 5 (azul) | Parcelas catastrales |
| `CRUCES_VIARIO` | 6 (magenta) | Cruces con carreteras |
| `CRUCES_HIDRO` | 7 (blanco) | Cruces con ríos |
| `INFRA_EXISTENTE` | 8 (gris) | Infraestructuras existentes |

## Salida

Un único archivo DXF por proyecto con todas las capas. Compatible con AutoCAD 2018+.
