## Project
`python
class Project:
    name: str
    alignment: Alignment
    buffer: Buffer
    settings: ProjectSettings
    municipalities: list[Municipality]
    parcels: list[Parcel]
    roads: list[Road]
    rivers: list[River]
    infrastructure: list[Infrastructure]
    results: AnalysisResults
    history: ProjectHistory
`

## Alignment
`python
class Alignment:
    segments: list[Segment]
    # Segment es una unión de: Line, CircularArc, Clothoid
    total_length: float
    stations: list[Station]  # PKs
`

## Parcel (Catastro)
`python
class Parcel:
    reference: str  # Referencia catastral
    area: float
    geometry: Polygon
    municipality: str
    province: str
    use: str  # Uso principal (rústico, urbano, industrial, etc.)
`

## Road

class Road:

# 008 — INTERFAZ GRÁFICA (GUI)

**Versión:** 0.0.1  
**Fecha:** 2026-07-09

## Framework

- **PySide6** (bindings oficiales de Qt6 para Python).

## Ventanas principales

1. **MainWindow** - Ventana principal con menú, barra de herramientas, panel de proyecto y visor de mapa.
2. **ProjectDialog** - Creación/apertura de proyectos.
3. **AlignmentEditor** - Editor gráfico del eje (puntos, curvas, clotoides).
4. **ResultsPanel** - Panel lateral con resultados de análisis.
5. **ExportDialog** - Diálogo de exportación Excel/DWG.

## Arquitectura MVC

`
Model (linpro.core) → Controller (linpro.gui.controllers) → View (linpro.gui.windows)
`

Los controladores orquestan los módulos de negocio y actualizan las vistas. Las vistas nunca importan módulos de negocio directamente.
