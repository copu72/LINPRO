# 002 — OBJETIVOS

**Versión:** 0.0.1  
**Fecha:** 2026-07-09

## Objetivos generales

1. Crear una librería Python robusta, tipada y documentada.
2. Automatizar la obtención y análisis de datos catastrales, municipales, viarios e hidrográficos.
3. Generar informes técnicos profesionales (Excel, PDF) sin intervención manual.
4. Generar planos DWG/DXF compatibles con AutoCAD y otras plataformas.
5. Mantener la independencia total de software propietario.

## Objetivos específicos v0.1

- [ ] Módulo geometry.Alignment funcional: definición de ejes con curvas circulares y clotoides.
- [ ] Sistema de PK con recalculo automático al modificar el eje.
- [ ] Buffer y análisis de afecciones sobre el eje.
- [ ] Descarga automática de datos catastrales (Sede Electrónica de Catastro).
