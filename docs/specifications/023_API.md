# 023 — API PÚBLICA

**Versión:** 0.0.1
**Fecha:** 2026-07-09

## Principios de diseño

1. **API minimalista y expresiva.** Pocas clases públicas, métodos bien nombrados.
2. **Inmutabilidad por defecto.** Las geometrías son inmutables una vez creadas.
3. **Validación en constructores.** Las clases validan sus parámetros al crearse, no al usarse.
4. **Separación de concerns.** La capa de API no contiene lógica de negocio. Solo orquesta.

## Ejemplos de uso esperados

```python
from linpro import Project
from linpro.geometry import Alignment
from linpro.roads import RoadAnalyzer

proj = Project("Línea 132kV")
alignment = Alignment.from_points([(0,0), (100,50), (200,200)])
proj.alignment = alignment
proj.run_analysis()
proj.export_excel("informe.xlsx")
proj.export_dxf("plano.dxf")
```
