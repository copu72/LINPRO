# 003 — GLOSARIO

**Versión:** 0.0.1  
**Fecha:** 2026-07-09

| Término | Definición |
|---------|-----------|
| **Alignment** | Eje geométrico de una infraestructura lineal. Secuencia de tramos rectos, curvas circulares y clotoides. |
| **PK** | Punto Kilométrico. Distancia acumulada desde el origen del alignment. |
| **Buffer** | Área de influencia alrededor del alignment (ej.: 50m a cada lado para una línea de 132kV). |
| **Clotoide** | Curva de transición cuyo radio varía linealmente con la longitud. Usada en carreteras y ferrocarriles. |
| **Afección** | Toda infraestructura, parcela o elemento que intersecta o está dentro del buffer. |
| **Catastro** | Registro administrativo que describe los bienes inmuebles rústicos, urbanos y de características especiales. |
| **INSPIRE** | Directiva europea que establece una infraestructura de información espacial. |
| **EPSG** | European Petroleum Survey Group. Código que identifica un sistema de referencia espacial (SRC). |
