# Project Glossary

**Versión:** 1.0
**Fecha:** 2026-07-09

| Término | Definición |
|---------|-----------|
| **Alignment** | Eje geométrico de una infraestructura lineal. Secuencia de tramos rectos, curvas circulares y clotoides. |
| **ADR** | Architecture Decision Record. Documento que registra una decisión técnica importante. |
| **Buffer** | Área de influencia alrededor del alignment (ej.: 50m a cada lado). |
| **Clotoide** | Curva de transición cuyo radio varía linealmente con la longitud. Usada en carreteras y ferrocarriles. |
| **Core** | Núcleo de LINPRO. Módulo base del que dependen todos los demás. |
| **EPSG** | European Petroleum Survey Group. Código que identifica un sistema de referencia espacial. |
| **FR** | Functional Requirement. Requisito funcional identificado (FR-XXXX). |
| **INSPIRE** | Directiva europea de infraestructura de información espacial. |
| **LINPRO Core** | Paquete base del que dependen LINPRO GIS, LINPRO CAD, LINPRO GUI, etc. |
| **PK** | Punto Kilométrico. Distancia acumulada desde el origen del alignment. |
| **Plugin** | Módulo independiente que extiende LINPRO. Cada uno en su propio directorio en /plugins/. |
| **UC** | Use Case. Caso de uso documentado (UC-XXXX). |
| **Uso principal (catastro)** | Clasificación de una parcela: rústico, urbano, industrial, etc. |
| **WFS** | Web Feature Service. Protocolo OGC para consultar datos geográficos. |
